# Waveforms

Simulation waveforms and transaction recordings captured during 
verification of the CFS Aligner DUT.

Each subfolder corresponds to a specific test run and contains:
- A waveform screenshot (`.png`) generated by [uvm-json-wave-viewer](https://github.com/username/uvm-json-wave-viewer)
- A trimmed simulation log (`.log`)
- Recorded transaction data (`.json`) for APB and MD agents
